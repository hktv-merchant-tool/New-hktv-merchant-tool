# HKTVmall B2B Merchant Tool v2

HKTVmall 商戶招商開發工具 — 靜態 HTML 版本，唔需要任何後端。

## 功能

- ✦ 文本生成器 — 招商計劃選擇、渠道與語言設定
- ◈ 佣金率與數據 — 16 個主類別、5,186 個分類碼
- ◉ 3PL 倉儲服務 — 收費標準、服務流程、聯絡方式
- ◎ FAQ 常見問題 — 24 條精選問題
- ⚠ 罰款清單 — 29 條違規條款

## 使用方法

直接用瀏覽器打開 `hktv_merchant_tool_v2.html` 即可。

## 部署到 GitHub Pages

1. 去 https://github.com/new 建立 repo，命名 `hktv-merchant-tool`（不要勾選任何選項）
2. 之後 push 代码：

```bash
git remote add origin https://github.com/hktv-merchant-tool/hktv-merchant-tool.git
git push -u origin master
```

3. 去 Repo Settings → Pages → Source 選 `master` branch
4. 等幾分鐘，你的網站就會喺 `https://hktv-merchant-tool.github.io/hktv-merchant-tool/` 上線
